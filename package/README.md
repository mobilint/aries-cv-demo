# ARIES CV Demo Package

This package is the deployable source bundle for the unified CLI-selected CV demo.

## Package Contents

The archive generated by `package/package.sh` contains:

- `src/demo/` source code
- `assets/` demo manifests, layouts, videos, and model files
- `Makefile` for package-local builds
- bundled `qbruntime` and `yaml-cpp` headers and libraries

## Requirements

- `g++` or another C++17-compatible compiler
- `make`
- `pkg-config`
- `libopencv-dev`
- Mobilint driver and runtime environment

## How to Build

Inside the extracted package directory:

```bash
make -j"$(nproc)"
cd build
./demo
```

The package `Makefile` builds the executable at `build/demo`.

## Runtime Configuration

The packaged application reads:

- `assets/weapon/config/demo.yaml`
- `assets/fire/config/demo.yaml`

When running `build/demo`, keep the package directory layout intact. The manifests resolve their own asset/model paths relative to the manifest file location.

## Demo Manifest

Edit `assets/<demo-name>/config/demo.yaml`.
`demo.yaml` points to the split setting files for layout, feeders, and models.

Key manifest fields:

- `layout_setting`
- `feeder_setting`
- `model_setting`
- `ui.overlay_style`

Current pipeline types:

- `yolo26_det`
- `yolo11_det`
- `yolo_anchorless_det`

## Shortcuts

| Shortcut | Description |
| --- | --- |
| `D` / `d` | Toggle FPS overlay |
| `T` / `t` | Toggle running time overlay |
| `M` / `m` | Toggle fullscreen |
| `C` / `c` | Stop all workers |
| `F` / `f` | Start all workers |
| `Q` / `q` / `Esc` | Quit |
